# 🌿 EcoMercat Lleida

## Objectiu del projecte
Plataforma ecommerce de productes ecològics locals de la Cooperativa EcoMercat Lleida.
Permet la venda online de fruita, verdura, làctics i conserves ecològiques de proximitat.

## Stack tecnològic
- **Frontend:** React + CSS3
- **Backend:** Node.js + Express
- **Base de dades:** MongoDB
- **Arquitectura:** Client–Servidor, API REST

## Instal·lació

```bash
# Clona el repositori
git clone https://github.com/EL-TEU-USUARI/ecomercat-lleida.git

# Instal·la dependències del backend
cd backend
npm install

# Instal·la dependències del frontend
cd frontend
npm install
```
